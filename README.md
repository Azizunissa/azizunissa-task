# Task Master - Progress Tracking Web App

Task Master is a simple, clean and interactive web application to add, organize, filter and track your daily tasks.  
It helps you manage tasks based on **priority** and **category** and even shows live completion progress.

### Features

- Add unlimited tasks
- Mark tasks as Pending → In Progress → Completed
- Task filters based on:
  - Status (Pending / In Progress / Completed)
  - Priority (High / Medium / Low)
  - Category (Work / Personal / Study / Other)
- Dynamic Progress Bar
- Beautiful UI with responsive design

### Tech Stack

- HTML
- CSS
- JavaScript (Vanilla JS)
- Font Awesome Icons

### How to Run

1. Download or Clone this repository
2. Open `index.html` in your browser
